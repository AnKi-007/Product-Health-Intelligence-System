# Monthly Product Health Intelligence System

## Project Overview
This project simulates an internal product analytics system used by a product-based company.
It ingests multiple real-world datasets, performs ETL, stores data in MongoDB, analyzes
monthly product health metrics, and uses LLMs to generate explainable insights.

## Tech Stack
- Python
- MongoDB
- Pandas
- Scikit-learn
- LLMs (for summarization and explanations)

## Project Structure
- data/        : Raw datasets (CSV)
- ingestion/  : Data loading scripts
- etl/        : Data cleaning and transformation
- analysis/   : Analysis and ML logic
- llm/        : LLM-based insight generation
- reports/    : Monthly reports
